# Vault 資料夾結構定義

初始化時需確保以下資料夾存在（不存在則建立，已存在則略過）：

## 必要資料夾

```
歷史紀錄/對話/
主題知識/
templates/
```

## 說明

| 資料夾 | 用途 | 特性 |
|--------|------|------|
| `歷史紀錄/對話/` | 對話筆記，按日期子資料夾 | 檔名格式 `[序號]_[概述].md`，子資料夾 `YYYY-MM-DD` |
| `主題知識/` | 知識筆記，按日期子資料夾 | 子資料夾 `YYYY-MM-DD` |
| `templates/` | 筆記模板 | 2 種類型，不直接存放筆記 |
