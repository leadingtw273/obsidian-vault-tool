# Vault 資料夾結構定義

初始化時需確保以下資料夾存在（不存在則建立，已存在則略過）：

## 必要資料夾（初始化時建立）

```
歷史紀錄/對話/
主題知識/
templates/
```

## 動態資料夾（由 archive / record-archive skill 按需建立）

```
歷史紀錄/YouTube/
歷史紀錄/Facebook/
歷史紀錄/文章/
歷史紀錄/文件/
歷史紀錄/網頁/
```

這些子目錄**不需要**在初始化時預先建立，由 `record-writer` agent 在第一次歸檔對應來源類型時自動建立。

---

## 說明

| 資料夾 | 用途 | 特性 |
|--------|------|------|
| `歷史紀錄/對話/` | 對話類來源記錄，按日期子資料夾 | 檔名格式 `[序號]_[概述].md`，子資料夾 `YYYY-MM-DD` |
| `歷史紀錄/YouTube/` | YouTube 來源記錄 | 同上，動態建立 |
| `歷史紀錄/Facebook/` | Facebook 貼文來源記錄 | 同上，動態建立 |
| `歷史紀錄/文章/` | 文章來源記錄 | 同上，動態建立 |
| `歷史紀錄/文件/` | PDF 文件來源記錄 | 同上，動態建立 |
| `歷史紀錄/網頁/` | 一般網頁來源記錄 | 同上，動態建立 |
| `主題知識/` | 知識筆記，按日期子資料夾 | 子資料夾 `YYYY-MM-DD` |
| `templates/` | 筆記模板 | 2 種類型：來源記錄、知識筆記 |
